# 🦺 SafeEye — AI Factory Safety Surveillance System

Real-time hard hat detection system using YOLOv8 deep learning model.
Monitors factory workers via webcam and triggers instant alerts on PPE violations.

## 🚀 Features
- Real-time hard hat detection via webcam
- YOLOv8 deep learning model (Hardhat trained)
- Instant violation alerts with screenshot saving
- Streamlit dashboard with live feed + violation history
- CSV violation log for compliance tracking

## 🛠️ Tech Stack
- Python, YOLOv8, OpenCV, Streamlit
- Computer Vision, Deep Learning
- Alert Pipeline with logging
